This project demonstrates the implementation of Inter-VLAN Routing using Router-on-a-Stick in Cisco Packet Tracer. VLANs help logically segment a network into isolated broadcast domains. Using a single physical router interface with subinterfaces, we enable communication between VLANs.

  1. Create VLANs on the switch and assign ports.
  2. Configure trunk port on the switch (e.g., switchport mode trunk).
  3. Configure router subinterfaces:
    • interface Gig0/0.10
    • encapsulation dot1Q 10
    • ip address 192.168.10.1 255.255.255.0
    • Repeat for VLANs 20 and 30
  4. Assign static IP addresses to end devices.

ðŸ“¡ What Happens in the Network

Each device is connected to its respective VLAN. When it sends traffic to a different VLAN, the packet is tagged, forwarded to the router via the trunk port, and routed through the correct subinterface. The router then sends it back to the switch for delivery to the target VLAN.

  4. Ping router subinterfaces to verify Layer 3 routing.
